### What I've built
- **[Vibook](https://vibook.app)** *(Founder)*. A Saudi events & ticketing platform: instant registration, secure QR tickets, organizer dashboards, on-site scanning. Grew to **1,200+ accounts, 20+ organizations, 25+ events**, generating **7,000+ tickets and 25,000+ visitors in its first 2 months.**
- **[Gaming+](https://gamingplus.gg)** *(Web Dev Lead)*. Led a team of 5 to ship the platform in Next.js: **13.9K+ visitors, 42K+ page views**, with a live ordering system handling **2,500+ orders.**
- **[KIRIX Media](https://kirix.sa)** *(Technical Lead)*. Built a fully automated photo-printing station (3+ min manual down to **10 sec end to end**) plus the company's full digital infrastructure: manager dashboard, employee database, auto certificate generation, and dynamic forms.
- **[Mdar](https://mdar-website.vercel.app)**. Real-estate financing comparison platform connecting users to banks to estimate eligibility.
- **[Karfeshha POS](https://karfeshha.vercel.app)**. Real-time POS web app with synced Order / Kitchen / Analytics views (Firestore).
- **Orod**. Short-video deals platform; businesses post promotions, users discover offers by category.
